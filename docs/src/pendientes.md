# Tareas Pendientes y Mejoras Futuras

Este documento detalla las funcionalidades que están en proceso de cambio o que se planean implementar para mejorar la experiencia de usuario en GutenAIR.

## 1. Funcionalidades del Editor

*   **Pegado Especial en el Editor**: Actualmente, esta función se dispara desde el panel lateral y reemplaza todo el contenido del capítulo. Se planea mover esta funcionalidad directamente al editor para:
    *   Permitir la inserción de texto parcial en cualquier punto del documento.
    *   Evitar la pérdida accidental de contenido al no destruir todo el `<body>` del archivo actual.
