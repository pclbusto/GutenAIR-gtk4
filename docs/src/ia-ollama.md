# IA con Ollama

La integracion con IA es experimental y esta pensada para conectarse a un servidor local de Ollama.

## Configuracion

Desde Preferencias, en la pagina IA, se configura la URL del servidor. El valor habitual es:

```text
http://localhost:11434
```

## Requisitos

Ollama debe estar instalado y ejecutandose fuera de GutenAIR.

Ejemplo:

```bash
ollama serve
```

La disponibilidad de modelos depende de la instalacion local de Ollama.

## Estado

La integracion puede cambiar. Trata esta funcion como asistencia experimental, no como parte estable del flujo principal de edicion.
